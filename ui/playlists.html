Playlist Generation

This playlist generator works in tandem with the query you input on the right side of this page. Given that query, it will look for the TrackID column that you've queried for, and gather the necessary information (filename, folder, etc.) to create a playlist for a specific platform.

The generator will create a playlist using everything after the first FROM in your query, so be careful not to include any logic in the SELECT columns themselves.
