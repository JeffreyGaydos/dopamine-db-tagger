Tagging

Explanation

Search for an existing tag or create a new tag using the inputs below. All tag objects on this side of the page can be clicked to be added to the current tack (if not already present). Tags with the ⨯ can be removed from this track using that button, but those tags will remain an "existing tag" until you use the right side of this page to edit or delete them.

Artist Tags are a special kind of tag that applies at the "Artist-Level", meaning the tag will be added to all tracks by that artist. The same tag can be applied at both the artist-level and the track level, but not on the same track.
